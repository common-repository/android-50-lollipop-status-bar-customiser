Use this plugin to change the Status Bar of Devices Running Android 5.0 Lollipop.

== Description ==

This plugin lets you customise the Status Bar of Android devices running Android 5.0 Lollipop. Simply activate the plugin, customise the settings, and away you go!

This plugin also lets you:
* Customise the Status Bar colour for your home page, Pages and Posts separately.
* Add an icon that appears on Android when the user is multiasking
* [New] Set colours for Post Categories (only the first colour will be applied where a post as multiple categories)

== Installation ==

1. Upload the `android-50-lollipop-status-bar-customiser` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set some Colours, Upload a Icon, and Save settings.
